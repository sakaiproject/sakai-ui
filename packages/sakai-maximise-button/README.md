# sakai-maximise-button

Handles the calling of portal.maximiseTool/portal.minimiseTool

## Installation

```bash
npm i @sakai-ui/sakai-maximise-button
```

## Usage

```html

<sakai-maximise-button></sakai-maximise-button>

```

## Linting and formatting

npm run lint
